# ASCII 로그라이크 생존 게임

Nethack, Stone Soup, Unreal World에서 영감을 받은 ASCII 기반 로그라이크 생존 게임입니다.

## 특징

- **ASCII 그래픽**: 클래식 로그라이크 스타일의 ASCII 문자 기반 그래픽
- **절차적 생성**: 매번 다른 던전이 생성됩니다
- **생존 시스템**: 배고픔, 갈증, 체온 관리 (Unreal World 스타일)
- **턴 기반 전투**: 전략적인 턴 기반 전투
- **아이템 시스템**: 음식, 물약, 장비 등 다양한 아이템

## 설치

```bash
# 의존성 설치
pip install -r requirements.txt

# 게임 실행
cd src
python main.py
```

## 조작법

### 이동
- **방향키**: 상하좌우 이동
- **hjkl**: Vi 스타일 이동 (Nethack과 동일)
- **yubn**: 대각선 이동
- **넘패드**: 8방향 이동

### 행동
- **g** 또는 **,**: 아이템 줍기
- **i**: 인벤토리 열기
- **x**: 둘러보기 모드
- **r**: 휴식 (체력 회복)
- **.** 또는 **s**: 대기

### 인벤토리
- **a-z**: 아이템 선택/사용
- **Shift + a-z**: 아이템 버리기
- **ESC**: 닫기

### 기타
- **?**: 도움말
- **Ctrl+Q**: 게임 종료

## 게임 심볼

```
@ = 플레이어          . = 바닥           # = 벽
g = 고블린           o = 오크           T = 트롤
% = 음식/시체        ! = 물약           > = 내려가는 계단
~ = 물               * = 불             ^ = 함정
```

## 생존 시스템

- **포만감**: 음식을 먹지 않으면 감소, 0이 되면 서서히 체력 감소
- **수분**: 물을 마시지 않으면 빠르게 감소, 포만감보다 더 급함
- **체온**: 환경에 따라 변화, 극단적인 온도는 위험
